# Resources

Curated source documents, datasets, and external references for Legal AI — organised by topic. Use these as knowledge bases for RAG systems, AI assistants, and demos.

## Subfolders

| Folder | Coverage |
|---|---|
| [`/gdpr/`](./gdpr/) | GDPR / AVG — EDPB guidelines, data protection documents |
| [`/ai-act/`](./ai-act/) | EU AI Act — official text, EDPB and AP guidance on AI |
| [`/ai-demos/`](./ai-demos/) | Demo datasets for RAG prototypes and Legal AI demos |
| [`/nis2/`](./nis2/) | NIS2 Directive — official text, Dutch implementation (Cyberbeveiligingswet) |
| [`/general/`](./general/) | Open Government (Woo), public procurement, ICTRecht publications |
