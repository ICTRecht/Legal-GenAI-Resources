# NIS2 Templates

Compliance documents for NIS2 Directive obligations (Directive (EU) 2022/2555), as implemented in the Netherlands via the Cyberbeveiligingswet — ready for AI-assisted completion or review.

## Planned templates

| Template | Description | Status |
|---|---|---|
| NIS2 Scope Assessment | Checklist to determine whether your organisation falls under NIS2 (essential / important entity) | Coming soon |
| Cybersecurity Measures Checklist | Article 21 minimum measures — gap assessment template | Coming soon |
| Incident Notification Template | Article 23 reporting template (initial / intermediate / final report) | Coming soon |
| Supply Chain Security Assessment | Vendor risk assessment per Article 21(2)(d) | Coming soon |
| Management Accountability Register | Board-level cybersecurity oversight documentation (Article 20) | Coming soon |
