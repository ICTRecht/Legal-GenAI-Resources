## Legal plugin voor Claude Cowork met de kennis van ICTRecht
# Plugins

Integrations that connect Claude to external legal data sources. Where a Skill shapes how Claude reasons, a Plugin gives Claude access to live external knowledge — court rulings, legislation, regulation texts, document systems.

## Available plugins

| Plugin | Integrates with | Status |
|---|---|---|
| [`ictrecht/`](./ictrecht/) | ICTRecht knowledge base, contract clause library | Available |

## Planned integrations

| Integration | Description |
|---|---|
| rechtspraak.nl | Direct access to Dutch court rulings without manual copy-paste |
| EUR-Lex | Full text of EU regulations and directives, including amendments |
| Wettenbank.nl | Current Dutch legislation |
| Document management | SharePoint / NetDocuments integration for direct document processing |

## Contributing a plugin

Plugins require a technical review before merging. Please [open an issue](https://github.com/ICTRecht/Legal-GenAI-Resources/issues/new/choose) first to discuss your integration approach.

See [CONTRIBUTING.md](../CONTRIBUTING.md) for format requirements.
